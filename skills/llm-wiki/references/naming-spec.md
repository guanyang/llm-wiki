## Naming Conventions

| Type | Naming Rule | Example |
|---|---|---|
| Entity page | Entity name, hyphen-separated | `Obsidian.md` |
| Concept page | Concept name | `RAG.md` |
| Summary page | `raw-` + abbreviated source path | `raw-articles-llm-wiki.md` |
| Comparison page | `A-vs-B-topic` | `LLM-Wiki-vs-RAG.md` |
| Synthesis page | Descriptive analysis topic | `index-and-log-scalability-analysis.md` |
